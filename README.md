## AIM
This Android application serves as a practical demonstration of using both Implicit and Explicit Intents to perform various common actions and navigate between activities. The application showcases the following functionalities:

1.Make a Call: Initiate a phone call to a user-specified number.

2.Open URL: Launch a web browser to display a user-specified URL.

3.Open Call Log: Display the device's call history.

4.Open Gallery: Allow the user to view images using a gallery application.

5.Set Alarm: Open the device's alarm clock application to set a new alarm (or show existing alarms).

6.Open Camera: Launch the device's camera application to capture an image.
